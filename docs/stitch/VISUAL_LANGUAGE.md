# Current visual language (baseline)

This is what exists today; you may evolve it, but keep the product feeling warm and food-forward.

## Notable colors used in UI
- Background: #fdfaf6
- Text primary: #451a03
- Text secondary: #92400e
- Primary accent: #d97706
- Darker accent: #b45309

## Typography
- Global font: Geist Variable (sans)
- Several headings use `font-serif` for a crafted feel

## Shape
- Very rounded corners are common (e.g. `rounded-[2.5rem]`, `rounded-[2rem]`)
- Soft shadows, often amber-tinted

## Motion
- Subtle page fade/slide transitions between tabs

Goal: keep warmth but improve hierarchy, density, and consistency.
