# BigBadMeals — product context

## What it is
BigBadMeals is a mobile-first meal and grocery planning app. Users can:
- Import recipes (text, URL, image) via Gemini extraction
- Browse and favorite recipes
- Auto-generate a weekly plan with AI
- Sync a shopping list from the latest plan
- Maintain household preferences in Profile

## Primary users
Busy home cooks planning a week; households with multiple eaters/constraints.

## Success criteria for the UX refresh
- Faster scanning + clearer hierarchy
- More consistent component language
- Better empty/loading/error states
- No behavior changes/regressions
